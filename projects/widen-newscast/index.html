Widen Newscast

A news aggregator that allows users to view news from multiple sources in one place.

Backstory

Random day in 2020, my friend Azfar Labib gave me this idea of creating a news portal, on which we will get news from different news sites. He had the idea, I had the skills(somewhat). So, I started working on it. Spent late nights trying out Python, Ruby(yes, Ruby, I started learning Ruby for it) scraping sites. But I was getting frustrated. With zero progress & luck, we gave up. Totally forgot about this project. After many years have passed. In DeepThought I've developed CPaaS, using the concept of extension, just like how VSCode works. One fine day I was thinking about CPaaS then I remembered about this project. I thought about the extension concept of CPaaS, then I thought, why not use it for this project. So, I started working on it. I've developed the core of the project just in 6 hours! With Prothom Alo extension, which gives the latest news from Prothom Alo. Thus, revival of Widen Newscast.

JavaScript Worker Threads

JavaScript Worker Threads act as separate instance created by main thread to accomplish resource heavy tasks.
e.g.: image resizing, counting sum of 1 to 1 billion number

Main Thread vs Instance Thread

When we start any website that runs JavaScript or a Node.JS server or run any JavaScript file with Node.JS, our JavaScript engine gets assigned with an ID, generally called PID. Think of it as IP Address, IP Address for our processes.
Whatever we run, it stays on the same PID, I mean it stays on main instance/thread.
What if we can take another IP Address and do some more task managed by our main thread?
That is instance thread. When we work with Worker Threads, we fire up multiple threads and assign tasks to those threads. All of them are managed by our main thread.

How is it different than async/await?

In async/await, we stay on main thread. In resource intensive tasks, we wait to finish that process then proceed.
In worker thread we use a different thread to accomplish that task so our main thread remains free.
