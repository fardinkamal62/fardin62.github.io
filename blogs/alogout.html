Auto Log out

My 'Iskul' project is almost a year old. It was done a long ago; some fixes needed to be done. One of them were: if someone logs in to account, auto log out from previous device; as I disabled multi-device functionality.

Until now one account can be signed in to more than one device. Which threw errors because of the Keep Me Logged In mechanism.

User info comes from database. For that error it shows Logged In User Not Found. Means it couldn't fetch the user data, the return value was just empty.

That's what I used to fix this. I checked the return value. If it was empty, expire the cookie.


The code:

    if(empty($row['name'])){header("Location: logout.php");};

I know I know I could've just iterate through database for cookie value but isn't it hassle free one-liner?
